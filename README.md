## 🧠 About Us

We are an on-chain development studio focused on building low-latency trading infrastructure and open-source tooling for the Solana and EVM ecosystems. We ship fast, publish what we learn, and play both sides of the market — attack *and* defense.

---

Our mission is to:

* Build **high-performance trading bots** and execution infrastructure

* Research **MEV, liquidations, and novel execution strategies** across chains
* Raise the bar on **wallet and trading security** for end users

## 📂 Featured Open Source Projects

* [solana-nova-sniper](https://github.com/savageStudios63/solana-nova-sniper) — A high-performance Rust sniper targeting Pump.fun and Raydium LaunchLab with Jito bundle execution, built-in rug filters, and a configurable TP/SL state machine.
* [mev-reaper](https://github.com/savageStudios63/mev-reaper) — An Ethereum MEV arbitrage stack: a gas-optimized Solidity executor contract (Foundry) paired with a Rust searcher daemon submitting via Flashbots.
* [rugx-scanner](https://github.com/savageStudios63/rugx-scanner) — A Next.js Solana token risk scanner grading authorities, LP status, holder distribution, and honeypot behavior into a 0–100 safety score.
* [onchain-alpha-engine](https://github.com/savageStudios63/onchain-alpha-engine) — An ML engine scoring freshly launched Solana tokens via on-chain signals — buyer quality, insider clustering, holder growth velocity.
* [hydra-tracker](https://github.com/savageStudios63/hydra-tracker) — A cross-chain terminal portfolio tracker built in Rust + ratatui, covering Solana, Ethereum, Base, and Arbitrum wallets in one TUI.
* [drainer-shield](https://github.com/savageStudios63/drainer-shield) — A browser extension that inspects wallet signature requests in real time and blocks common drainer patterns on Solana and EVM.
